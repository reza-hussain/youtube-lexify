# YouTube Lexify — Brand Guidelines

## 1. Brand Overview

**Product Name:** YouTube Lexify  
**Tagline:** Understand Every Word. Instantly.  
**Category:** Educational / Productivity / Language Learning  
**Platform:** Chrome Extension + Web Dashboard  

**Mission:**  
Empower users to understand spoken content on YouTube in real-time through intelligent subtitle interaction and contextual definitions.

**Vision:**  
Become the default language companion for video-based learning.

## 3. Visual Identity

### 3.1 Design Philosophy

Primary Style: **Liquid Glass (iOS26-inspired)**

Principles:
- Translucency
- Frosted blur
- Soft glow
- Depth via shadows
- Motion-based feedback
- Rounded geometry
- High contrast typography

---

### 3.2 Color System

#### Primary Palette

| Name         | Hex        | Usage |
|--------------|------------|-------|
| Glass White  | #F7F9FB    | Base background |
| Blur Gray    | #E2E6EB80  | Glass panels |
| Sky Blue     | #4DA3FF    | Primary CTA |
| Neon Cyan    | #38F2FF    | Highlights |
| Midnight     | #0F172A    | Dark mode base |
| Frost Blue   | #A5CFFF    | Accent |

Opacity Standard:
- Glass panels: 70%–85%
- Hover layers: 60%

---

### 3.3 Typography

Primary Font:
- Inter / SF Pro (Fallback: system-ui)

Hierarchy:

| Type      | Size | Weight |
|-----------|------|--------|
| Heading 1 | 32px | 700 |
| Heading 2 | 24px | 600 |
| Body      | 15px | 400 |
| Caption   | 12px | 400 |

Line Height: 1.4–1.6

---

### 3.4 Icons

Style:
- Outline + Soft Fill
- Lucide / Heroicons style
- 1.5px stroke

Icon Color:
- Default: #CBD5E1
- Active: #4DA3FF

## 4. Layout System

### 4.1 Chrome Extension UI

Zones:

1. Toolbar Popup
2. Subtitle Overlay
3. Definition Tooltip
4. Settings Panel

Structure:

- Max Width: 380px
- Radius: 18px
- Padding: 16px
- Shadow: 0 20px 50px rgba(0,0,0,0.15)

---

### 4.2 Video Overlay Layouts

#### UI Mode 1 — Compact Box

- Positioned above subtitles
- Width: 80%
- Height: auto
- Floating glass card

#### UI Mode 2 — Expanded Panel

- Right side panel (25% width)
- Sticky
- Divided vertically

Top:
- Live subtitles
- Auto-highlight

Bottom:
- Definition viewer

---

## 5. Motion & Animation

Framework:
- CSS + Framer Motion style

Timing:
- Micro: 120ms
- Small: 180ms
- Major: 280ms

Easing:
- cubic-bezier(0.22,1,0.36,1)

Examples:
- Fade + Blur on open
- Glass shimmer hover
- Elastic toggles

---

## 6. Interaction Rules

### Hover Behavior
- Delay: 200ms
- Cancel on scroll
- Fade-in tooltip

### Click Behavior
- Toggle persistent panel
- Save word automatically

### Pause Logic
- Auto pause after 500ms focus
- Resume on close
